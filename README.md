# Fulcrum Engine (FulcrumEngine)

An open-source, modular, cross-platform .NET game engine, Rendering engine powered by the Veldrid (Vulkan) rendering backend and released under the GPL-v3 license.

## Introduction
FulcrumEngine provides a GameObject/Component framework, multi-threaded rendering, a scene system, an NAudio-based audio subsystem, and BEPUphysics v2 integration. Its long-term goal is to build an extensible, free, and open game-development platform.

## Features
- Component-based GameObject architecture  
- Vulkan rendering (Veldrid)  
- Multi-threaded rendering/update  
- ImGui debug UI  
- OpenAL-based 3D audio  
- BEPU v2 physics integration  
- Modular, replaceable, extensible architecture
